# Mistress - load testing tool for server-side applications
This distribution is **statistics server** component of Mistress.

## Setup
	cd mistress-stat
    virtualenv --no-site-packages venv
    source venv/bin/activate
    pip install -r req.txt

## Usage
	cd mistress-stat
	source venv/bin/activate
    ./run
